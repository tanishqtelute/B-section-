The first version of HTML was written by Tim Berners-Lee in 1993. Since then, there have been many different versions of HTML. The most widely used version throughout the 2000's was HTML 4.01, which became an official standard in December 1999. HTML stands for Hyper Text Markup Language. HTML is the standard markup language for creating Web pages. HTML describes the structure of a Web page. HTML consists of a series of elements. HTML elements tell the browser how to display the content. HTML was invented by Tim Berners-Lee, a physicist at CERN, in 1990. He aimed to create a simple way to share and access documents over the Internet. Since its inception, HTML has evolved significantly, becoming the foundation of web development
